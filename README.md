2020-12 POJ1000：A+B Problem - 解题报告（张三）

## 内容

### 题目大意
计算两个数的和
### 解题思路
考察基本的C语法，POJ这道题不涉及多组数据，直接输出即可。而使用 EOF 判断文件结尾也没有问题。

### 复杂度分析
此题只执行一次加法运算，时间复杂度为 `O(1)`。没有涉及数组操作，空间复杂度也为`O(1)`

### 算法资料与心得体会
该题目无算法

### 标程
```C++
#include <stdio.h>

int main()
{
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}
```
